## commands
for running the dev server
```
npm run dev 
```
for running the tailwind css CLI
```
npm run tailwindcss 
```
for opening the drizzle studio 
```
npm run studio 
```
for pushing the schema to the database
```
npm run db:push 
```
for seeding the database
```
npm run db:seed 
```
